Description official description

Apprentice is the story of Mortimer "Pib" Pibsworth, an apprentice to a great wizard. Most days involve sweeping floors for the Master or collecting spell components, but today is different. Pib is going to cast his first spell.

The game details the story of Pib preparing for the spell. The gameplay consists of humorous multiple choice dialogues and inventory based puzzles.

A short (and unfinished) but extremely beautiful little AGS freeware adventure game

The Good
Another AGS game worth noting in my book that is worth the time and downloading. The other AGS game better than this (way better) is 5 Days a Stranger.

The game has exceptional smooth cartoon-like graphics to begin with, humor-like conversations similar to Monkey Island.

The music in the game is also incredible...especially in the scene with at the barn (see screenshots), the music there is exceptionally beautiful. I would say so far that the particular music background in that scene is more beautiful than the whole game itself.

The Bad
This game really needs a lot of work. Conversations needs a major upgrade as well as the humor. Unfortunately, the most funniest scenes is during the intro cut-scene but hardly found in the game itself which is kinda lame. One would think that the intro (hence the word) only introduces a small part of the 'better things' of the game that a gamer would look forward to. Unfortunately in this game, it kinda degraded the concept.

The crappiest thing about the game is that it's technically isn't finished. I don't know if there's a newer version of the game that added more scene's, since this game is only more or less 9 screenshots long.

The bad thing about an 'unfinished game' is that that you can finish this game in a couple of hours (or minutes) if solving puzzles is your thing. Therefore, the storyline is kinda weak if not non-existent at all. This is a real drag because I saw a lot of potential in the game, but when you want more gameplay, you realized that you've finished the game. Oh well, what'dya expect from a freeware game?

The Bottom Line
A small adventure game with a lot of potential ahead of it. Would really love to see the whole game when it's fully completed. Totally worth the download.

Windows · by Indra was here (20633) · 2004
