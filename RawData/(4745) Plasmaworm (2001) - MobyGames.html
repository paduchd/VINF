Description official description

Plasmaworm is an arcade game for all ages. It is a modern relative of early computer games such as Snake, TRON Light Cycles and a rare gem called Pizza Worm.

The game includes 20 different levels of increasing difficulty, playable in single player as well as two-player modes. You can also create your own levels and music using the built-in editors.
