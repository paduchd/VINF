Trivia

CD32 changes

There were slight changes made to CD32 version of "Alien Breed Special Edition". Apart from controls which were redirected to joypad, offering several different usage of 4 buttons, there were also some modifications in the Intex Network System. The player was given a possibility to enter an access code (using directional movements of the joypad when selecting letters) that allowed to start the game from further levels.
