Trivia

Apple IIGS version

A beta version exists of this game for Apple IIGS computer and remains 'unreleased'. Rumor has it that the game was ported without permission and original publishers were not interested in supporting Apple IIGS platform any longer.

Nintendo DS version

An unofficial port of the game has been developed for the Nintendo DS. Check out Links/Searches.
