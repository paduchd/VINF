Description

This is a fan attempt, using the program DarkBASIC and from memory, to recreate the 1976 arcade game from Exidy on the home computer. Use the "5" key to enter coins for more time and use "1" to start the game. The arrow keys drive the right car, the "z', 'x', 'c', and 's" keys control the left. Drive over the pedestrians for points. When they are hit, they scream and turn into tombstones.

Trivia

Development

This game was made from memory as Doc (the developer) didn't have a working copy of the arcade Death Race at the time. As such, the game may not play and look exactly like the original.
