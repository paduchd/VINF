Description official description

Cosmic Carnage is a one-on-one fighting game set in outer space. The story begins as a prison barge is hijacked by it's cargo in an escape attempt. The resulting chaos caused the barge to crash into a military cruiser. The impact caused extensive damage to both ships. There were only 8 survivors, 4 from each ship. What's worse, there's only one escape pod. You control one of the 8 survivors, and have to finish off all the others to escape with your life.

The gameplay in Cosmic Carnage is similar to Street Fighter 2, but with a much slower pace. Some of the fighters can be customized with different armor combinations, allowing them access to different special moves.

Slow action 2D fighter

The Good
The promise of a great fighting game on a brand new system with graphics as good as an arcade game. Note the word "promise".

The Bad
All hype that came down to a slow game with minimal game play. Sure it looks great but the game play leaves a lot to be desired. Anybody paying full price for this along with the launch price of the 32x would have been severely disappointed.

The Bottom Line
A simple street fighter type fighting game with aliens. One of the bottom games for the 32x

SEGA 32X · by gametrader (208) · 2004

Trivia

Version differences

Cyber Brawl, the Japanese version of the game, contains three human soldier characters in place of the US version's aliens. Apparently, these characters were made aliens for the US release to avoid complaints of graphic violence.
