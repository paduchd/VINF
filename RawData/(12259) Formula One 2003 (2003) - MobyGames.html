Description official description

This game is an officially licensed product of the FIA Formula One 2003 championship. It features all the drivers, tracks (including changes), teams and the new qualifying rules and point tables. You can fine-tune almost every aspect of the chosen car to create the best setup for the current track. You can set gear ratios, set front and rear air foils/wings etc etc. This time around, you can also set the brake bias.

The game has a single player and a multiplayer mode. Some settings such as gearbox, screen layout, and cockpit camera mode can be customised separately for each player. In addition to the standard controller, which has four non customisable configurations to choose from, the game also supports the Speedster 2, Driving Force and GT Force steering wheels.

The single player game has Arcade Race, Arcade Season, Custom Race, Race Weekend, 2003 World Championship and Time Attack modes of play. The multiplayer game has Arcade Race Player Vs Player, Custom Race, Hot Seat and Time Attack options. The Multiplayer Arcade Race and Custom Race options are split screen games while the players take turns to race in the other two options.

The game also has a Spectator Mode, where the game runs all the cars and the player can sit back, relax, and watch the action

Trivia

Cover

Every country in Europe this game was released in, featured a different cover showing their national driver(s) or team in action. For instance; The Spanish cover shows Fernando Alonso (Renault), the German cover shows Michael Schumacher (Ferrari), the Italian version shows Michael Shumacher (Ferrari) and Giancarlo Fisichella (Jordan-Ford).
