Description official description

Links: Championship Course: Castle Pines Golf Club is a golf course add-on compatible with the following games:

  • Links: The Challenge of Golf Links 386 Pro Microsoft Golf

The game was designed by Jack Nicklaus, with the game's location near Castle Rock, Colorado, USA; the longest course in the PGA tour.
