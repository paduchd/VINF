Description official description

Reader Rabbit is an educational game for children 3 through 7 designed to help reading and spelling skills. There are four different word games to play of increasing difficulty: a word sorter, a picture labeler, the word train, and a variation of the game memory.

One of the first games I've ever played.

The Good
When I was 2 years old (that'd be 1989) my dad got this game for his old Commadore 64.

THAT'S 64 KILOBYTES OF PURE COMPUTING POWER, BABY!!!!

Anyway... this was one helluva fun educational game for a kid, and it had alot of fun activities, like organizing words, and memory games, etc.

The Bad
Well the game is just terrible for graphics by todays, but the CGA is really nostalgic for me.

The Bottom Line
I can't really recommend this game for anybody's kids nowadays, because the controls are a little clumsy, and the graphics and sound would sound pathetic on most modern computers...

Not like you could actually get the game working, except maybe with DosBox.

DOS · by bombad_jedi (4) · 2005
