Description

This is the final game in the Exile series. This is not a continuation of the plot of the series however - it is a program that allows you to make your own Exile games - although it includes three premade scenarios.

Trivia

Open source release

In May 2007, Jeff Vogel released the source code of the game, along with all data assets, under the terms of the Common Public License. In June 2007, the license was changed to the GNU General Public License, Version 2. Since then the game has been ported to run natively under 32-bit Windows, Mac OS X and Linux.
