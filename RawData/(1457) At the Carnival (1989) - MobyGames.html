Description official description

The first and only game using "The Puzzle Gallery" engine, for which there were supposed to be multiple Puzzle Disks. Extremely similar in style to Are We There Yet? and The Fool's Errand, "At the Carnival" includes word searches, jigsaws, block puzzles, word squares, crosswords, jumbles, cryptograms, mazes, and more.

Each puzzle is set at and somewhat related to a different attraction at Hazard Park. There are total of 40 attractions, and 180 puzzles

Trivia

Long Beach mummy incident

The game contains a first hand account of the, oddly true, "Long Beach mummy" urban legend. Cliff Johnson, the creator of this game, was working for the park when they discovered that the prop in question was, in fact, a cadaver.
