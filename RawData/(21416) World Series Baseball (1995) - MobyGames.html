Description official description

This version of World Series Baseball features the MLB and MLBPA licenses, three different baseball parks, and CD-quality sound. Gameplay modes include:

  • Exhibition
  • Pennant Race
  • Playoffs
  • All-Star Game
  • Home Run Derby
