Description official description

Tom Clancy's Ruthless.com is a game in which the player runs an up-and-coming computer software company, and tries to make it in the industry by any means necessary. As the name implies, the player is allowed to be as unethical as he can be. The game includes several scenarios, including the one called "Take Down EvilSoft", in which the player's company creates an unholy alliance with other companies to take down a corporation similar to Microsoft. The game has a board-like layout.
