Description

Ultimate Trainz Collection contains an updated 1.5 version of the original Trainz with new features including new scenarios, multiple cab views and dual cab controls, a manager for custom content, new track side objects and new locos such as the GG1 and TGV. The new scenarios are Biriburra Yards, Banks Heath, East Coast Sunlander, Robe River Iron, Alcopulse Railroad and Blue Sky Steel. With internet access, you can read the in-game Trainz eXchange Magazine where news, reviews and tips are shared.

The package also includes an Ultimate Trainz Collection edition of the Paint Shed program, a design tool to repaint the locos.

Trivia

Trainz Railway Simulator: Ultimate Collection

The UK release comes on two CDs. CD1 auto-loads and installs Trainz v1.3, when complete the install process exits and displays the game's release notes together with a message that CD2 contains the latest service pack and will update Trainz to version 1.5 (Build 404), Paintshed is also on CD2
