Description official description

Bon Voyage is the sixth edition in expansion pack for the original The Sims 2. Similar to the Vacation add-on for the original The Sims, the focus is on travelling. It requires the original game to play and offers 3 premade vacation destinations with distinct themes (Three Lakes, Takemizu Village and Twikkii Island) along with the ability to create new ones.

At the new locations, player can build and decorate vacation homes, there are new accessories and character can now walk to the community lots instead of using a vehicle. New NPCs include Bigfoot, hotel staff, fire dancers, ninjas and pirates.
