Description

First-person 3D accelerated space fighter combat game, played in enclosed arenas connected by teleports. Real physics down to friction, momentum, and inertia. Six vehicles, 11 weapons/powerups, force feedback, 3-D sound. Network play supports 63 players on a LAN and 24 on 28.8 modems. Release version ships shrinkwrapped with Linux & Windows binaries.

Trivia

Budget

The game was developed by independent game developer Aegis Simulation Technologies on a $35k budget with no outside funding.

Linux

This is the first game to ship with Windows 95/98/NT and Linux binaries on the same CD out of the box.
