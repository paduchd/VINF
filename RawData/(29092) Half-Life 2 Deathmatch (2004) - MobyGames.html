Description

Half-Life 2: Deathmatch is the stand-alone multiplayer version of Half-Life 2. The original game did not contain a multiplayer part, and it was released later. The main goal of the game is to annihilate your opponent(s) in a free-for-all multiplayer battle.

The game features the famous HL2 gravity gun which turns random objects into deadly ammunition, making it the first online FPS game to heavily utilize physics simulation for gameplay. Besides the standard HL2 weaponry it adds the Stunstick and the Selectable Lightweight Attack Munition (SLAM). The game features a Deathmatch and a Team Deathmatch scenario (Combine vs. Rebel). This game is often included with various Half-Life packs, both offline and online.
