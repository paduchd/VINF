Description

This is a compilation of Premier Manager 3 and its Multi-Edit Construction set. PM3 has been updated to 1995-1996 season data, with promoted and relegated teams moved divisions and player transfers activated - this means that previous saved games aren't compatible.

Trivia

Copy protection

Although the basic code wheel copy protection system was the same, it was updated to make it more secure and try and prevent a 100% cracked version from appearing. It took over ten years for a 100% cracked version to eventually appear.
