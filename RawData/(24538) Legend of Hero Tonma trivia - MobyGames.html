Trivia

Canceled NES release

A port of the game was being developed by Japan System House Co., Ltd. for the Famicom. It was suppose to be released in October of 1990 but it was never released.

Windows release

The game was one of the eighteen arcade games included in the compilation Irem Arcade Hits.
