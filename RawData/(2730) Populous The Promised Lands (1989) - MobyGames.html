Trivia

Disk format

The original release of The Promised Lands is one of a few programs for which the Amiga and Atari ST versions were released on the same disk. The disk contains a custom boot sector that can be read by both systems, while the rest of the disk has one side in standard ST and the other in standard Amiga format.
