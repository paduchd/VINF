Trivia

Cancelled US Game Boy release

The Game Boy version was planned to be released in the US but was ultimately cancelled.

Game Boy version

The Gameboy version replaces the swimming event with a free-fall fighting event in which two characters jump from a building and try to defeat each other before they hit the ground.
