Description official descriptions

Shanghai is a puzzle game using MahJongg tiles where you slide available matching tiles off the stack to reveal more tiles.ã€€Slide all 144 tiles off the board and you win the game.

This game includes four versions of the classic Shanghai game: The Great Wall (Shanghai meets Tetris), Beijing (slide rows to make matches), Action Shanghai (clear tiles quickly before more appear), and Classic Shanghai (either regular or face down for an extra challenge).

So far, I have not been able to find any substitute for this great game.

The Good
The games vary. Beijing, Great Wall, Action Shanghai, Classic -- all in one. I have enjoyed it so much but now, I can no longer play because the game is not compatible to my current platform, i.e. Windows XP Professional. I have been searching to find another version of Shanghai Great Moments, or at least can provide equal enjoyment that Shanghai has. Nothing is as good as Shanghai Great Moments.

The Bad
Nothing. I love all of it. I don't get tired of playing Shanghai Great Moments over and over again. Unfortunately, when I have to upgrade my PC platform to Windows XP Professional, my most treasured CD ROM of Shanghai Great Moments is unusable.

The Bottom Line
A MUST HAVE game. This is the game that last forever and you won't regret it. It is a collectible item.

Windows 3.x · by daisy subakti (1) · 2003
