Description official description

Get Set To Learn features five fun and challenging educational activities for children 3 and 6. Each activity is presented by a different muppet from the T.V. show.

Reviews

Players

Average score: 4.0 out of 5 (based on 6 ratings with 1 reviews)

Excellent arithmetic counting skills for age 3-6 or so.

The Good
A fair bit of action with The Count, Cookie Monster, Zoe, Big Bird etc. Music/Sound effects by Miles Ludwig and his gang of audio engineers. Great gameplay for children younger than around 7 or 8. Teaches counting skills. Can be copied from 3 directories and executed vie the .exe

The Bad
Could use more movement, faster synching animations. The games performance capacity was suppressed to be identically playable on slower computers. In spite of these problems it is interesting and creative, and the programming was skillful for it's time.

The Bottom Line
Will become dated and interesting over time as an early production 2D cartoonish Sesame game. This is the second in a series that was produced in-house before the name change from Children's Television Workshop to Sesame Workshop and the commercialization/coorporitization of the company, and is therefore of great historical value (Let's Make A Word, Search and Learn Adventures, Elmo's Preschool, Grovers Travels....late 1990's vintage.

Windows · by Todd Marshall (7) · 2003

Trivia

This game was going to feature Elmo, but when Sony Wonder started working on a line of games featuring Elmo then Creative Wonders chose to make a game with all the other characters from Sesame Street.
