Description official description

The license for this game was acquired from the German blockbuster comedy of the same title by a very popular German comedian called Michael "Bully" Herbig. The movie is parody on SF series/movies like Star Trek and Star Wars. The story of this point-and-click adventure is close to the movie's plot: the player consecutively takes over the roles of Captain Kork, Mr. Spuck and Schrotty and has to solve many riddles and tasks such as activating the spaceship's own musicbox so that the crew can practice for the "Miss Waikiki" dancing contest. The main story features some mini games like dancing. The characters are controlled by mouse only, using the left mouse button to move the character to the indicated places, start dialogues with other characters or use items. If an item or a person is clicked with the right mouse button, a context menu with the respective information appears.The actors have given the game characters their voices and have recorded over 1600 different spoken sequences.

Trivia

Reviews

The review copies sent to the press only included the first two thirds of the game because the movie production company did not want to leak information about the movie's ending. Because of this, at least a few publications chose to postpone the review until the full version was released.
