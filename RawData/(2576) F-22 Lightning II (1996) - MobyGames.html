Description official descriptions

Yet another simulation of the yet to enter service F-22, probably the most advanced fighter of the world. As with most Novalogic games, however, the emphasis here is on action and graphics, rather than in simulating the real thing.

The game offers a standard series of campaigns as well as a quick mission mode and multiplayer support for deathmatch and cooperative games using a feature similar to the "spawn" feature found on WarCraft 2.

Trivia

Patch

1.01.00.18 patch updates:

This upgrade adds the following features:

Windows 95 shortcuts:

Adds shortcuts to F-22 in the Start Menu and on the Desktop.

Chat mode:

Chat mode is accessed by pressing the "T" key. A composition overlay will appear. When there are more than 2 players, a receiver for the message must be chosen. Chat mode is not available on all screens.

Additional views:

Look Left - Keyboard Forward Slash

Look Right - Keypad Asterisc (star)

Look Up - Keypad Minus

Look Back - Keypad Plus

More Quick Missions:

Added Campaign 5 missions to the Quick Mission menu.

More joysticks supported:

Native support for CH Combat Stick, Sidewinder 3D Pro I & II.

This upgrade also contains the following maintenance fixes: Video:

VESA 1.0 override (this does not correct installation problems)

Sound:

Static in digital sound and music

Crashes caused when trying to use daughterboards

Modems:

F-22 will now recognize modems using IRQ's higher than 7
