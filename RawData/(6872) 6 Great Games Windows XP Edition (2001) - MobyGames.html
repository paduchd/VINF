Description official description

6 Great Games: Windows XP Edition is a multi-pack which contains full versions of:

  • Creatures 3
  • Microsoft Golf 1998 Edition
  • Missile Command
  • New Vegas Games
  • Platinum Pinball
  • Tomb Raider III: Adventures of Lara Croft
