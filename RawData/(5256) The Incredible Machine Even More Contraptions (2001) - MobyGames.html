Description official description

In The Incredible Machine: Even More Contraptions, gameplay is identical to Return of the Incredible Machine: Contraptions. The game contains 250 puzzles (although the 50 tutorial puzzles are identical to those in the previous game).

The latest instalment of The Incredible Machine for Mac OS X

The Good
Before installing the game on my Power Mac G4, I noticed two installers. One for Mac OS 9, and the other for the PowerPC version of Mac OS X. It was Sierra and Dynamix' idea to make it Mac OS X compatible since it is the first game to support it back in 2001 when the operating system and game are both released. It also includes music from The Even More Incredible Machine and The Incredible Machine version 3.0 that has made its way back into the new version in which I heard way back when.

The Bad
The MIDI versions were still heavily optimized for the Sound Blaster 16 (for those who are still and have been using DOS and Windows 95 for a long time) from the DOS versions of the early Incredible Machine titles. Some of the music from the early titles has been missed in this version. Kind of odd.

The Bottom Line
If you want some wacky fun, then I definitely recommend it! It even has new levels that were exclusive to the game. And for those who still have their PowerPC Macs running Mac OS X out there would really like to take this out for a spin!

Macintosh · by Katie Cadet (9959) · 2015
