Description official descriptions

Gladiator resembles Gauntlet, set in medieval times. You must travel the forest and kill off all the monsters, collecting treasures en route. You can select from seven different classes of characters, each with their own strengths and weaknesses in areas such as strength, dexterity and armor.

Trivia

Development

Gladiator was written completely in Watcom c++ 10.0B.

Source code release

In 2002 Forgotten Sages released the source code to Gladiator under the GPL; a team of fans, operating under the name Snowstorm Entertainment, have since mobilised on Sourceforge, "a collaborative software development management system", to further develop the title as Openglad with ports to Windows, Linux, FreeBSD, OpenBSD and BeOS.
