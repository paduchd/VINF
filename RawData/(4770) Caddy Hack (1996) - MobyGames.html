Description

Caddy Hack is a miniature golf game with a horror theme to it. You get to play in 9 holes each with a different theme and each with multiple paths, levels, etc.

Graphics consist of digitized images serving as the backgrounds, with a transparent putter which you can rotate all around the ball.
