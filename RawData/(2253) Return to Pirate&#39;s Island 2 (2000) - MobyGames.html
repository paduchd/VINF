Description official descriptions

If you remember "SAY YOHO" you won't need to read any further. This is Scott Adam's return to the wonderful world of interactive fiction, otherwise known as text adventure, after a long hiatus. In this game you don't spend a lot of time using your mouse, all commands are text. RTPI2 is a full-sentence text adventure game with sound effects for a Windows based personal computer running either Windows 95, 98, ME, NT or 2000.

In this game the object is to collect a series of treasures and return them to a central place for scoring. You'll work with, or against, a pirate, of course. In order to collect all the treasures you'll need to be creative and there is an extensive help system that will provide you hints to help you accomplish your goals.
