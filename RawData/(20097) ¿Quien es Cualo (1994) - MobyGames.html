Description

Board game in which there is a character selected by the computer for you to guess.

In the screen you see a board with different cards, each one representing a character, the game gives you hints like hair color, sex, eyes color, etc. and you must guide yourself with them and try to guess who the mysterious character is. You can also switch roles with the computer and make it guess which character you selected by giving the CPU the same kind of hints.

The characters are from the adventure game El principio del fin by the same developers.

Trivia

The portraits of the characters were going to be "different" at first, but the developers thought that some of them were too insulting (the characters were teachers of the game makers) and decided to use normal portraits as seen in the game El principio del fin. Although some of these offensive portraits can be seen in the title screen...
