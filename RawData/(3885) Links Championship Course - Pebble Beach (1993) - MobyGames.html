Description official description

An add-on course compatible with Links VGA, Links 386 Pro, and Microsoft Golf. Play the course created by the author of Morse Code and made famous by Bing Crosby's "Clambake" in 1947.

Trivia

Extras

As with other courses in the Links series, a scorecard was included in the package.

Packaging

Links: Championship Course - Pebble Beach is one of the only (possibly the only) add on course to feature the "Links Classics Edition" logo in place of the usual SVGA logo. The package was complete with raised, gold colored lettering, and gold colored disk labels.
