Description

Invaders 1978 is a clone of Taito's Space Invaders, modeled to closely resemble the original arcade version. The player is at the bottom of the screen facing upwards, moving left and right to shoot the row of aliens as they move left and right. Aliens at the top of the screen are worth more points.

This version has wider graphics than the original, and is a little quieter, reflecting the 'no frills' design philosophy. To make it a carbon copy, there is no high score table and keys cannot be remapped. The game was originally available through mail order, and later became freeware.

Trivia

Business model

With the v02 version in December 2000, the game's distribution model was changed from mailware to freeware. Also, the source code was included, along with some small fixes. The source is for QBASIC and it compiles with QuickBasic 4.5.

Development

The author reports the game was written for Peter Cooper's 1996 competition (among readers of the comp.lang.basic.misc Usenet group) to code a clone of any retro arcade classic in any version of BASIC. This game took first prize, albeit out of only four contestants.
