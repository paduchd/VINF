Description official descriptions

If you like the other Army Men games, then you'll like this one too. Similar gameplay with new weapons and an entire new race of plastic soldiers. Help Sarge and Tina Tomorrow save the Earth from alien invaders.

Trivia

Extras

As part of the promotion for this game, two plastic army men were included with the original packaging. Each package had one green and one tan army man, but their type (flamethrower, rifleman, courier, etc) changed with each package.
