Description

Drop Mania is a falling block puzzle game. The main idea of the game is to drop down pieces and make group by colors, when player drops an explosion brick, it explodes all the same color blocks that are in a group. The more blocks you get by one explosion, the more points you get. You can also get combos or chains. If two different color groups explode at same time, you will get a combo bonus, you will get chain bonuses if explosions happen in a row.
