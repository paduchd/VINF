Description

Master Mind is a simple clone of the classic board game with the same name. A color code is hidden at the top. You have 10 tries to figure it out by trying different color combinations. Black and white pins tell you if how many correct colors you have - white means that a pin is in the right place, black that the selection does include that color pin, but in a different place. By using logic you can crack the code!
