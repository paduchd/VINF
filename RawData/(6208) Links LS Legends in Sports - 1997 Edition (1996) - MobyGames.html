Description official descriptions

The 1997 Edition of the popular Links golf simulation. New features include higher resolution graphics (up to 1600x1200 in 16.7 million colors) with a new terrain rendering engine, and improved ball flight physics. Three new courses are featured, Latrobe Country Club, Kapalua Village Course, and the Kapalua Plantation course. Also included is a video tour of Latrobe Country Club, and a virtual reality tour of Arnold Palmer's office and workshop.

Trivia

Windows compatibility

Although Links LS 1997 was a DOS program, Access Software made a good attempt to make sure it was compatible with Windows 95. Included with the program was the "Links LS system monitor". This was a Windows 95 program which would attempt to detect any running windows programs or system configurations that would cause the game to crash. If everything looked OK, it would run the game. (This system monitor appears to be incompatible with Windows versions greater than 95.) Two versions of the install program were included as well, one for DOS and one for Windows.
