Trivia

Mods

There has been several mods released by 3rd party developers for the original 'Joint Operations Typhoon Rising' game, however not all work with the Combined Arms release.

At least one of the 3rd party mods for this game, 'Advanced Warfare 2', will not install if you have the disk-less 'Gold' version of the game, as it needs an original CD for source verification.
One other of the 3rd party mods:' International Conflict' has been fixed to allow installing with the disk-less 'Combined Arms Gold' release.
