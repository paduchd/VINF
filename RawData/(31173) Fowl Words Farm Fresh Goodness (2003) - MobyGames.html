Description official descriptions

In the word game Fowl Words / Chicktionary the player must find as many words as possible using the 7 letters given - either against the clock or not.

At the beginning of each round, you have to choose the bonus you try to get. The higher the bonus, the more points you get - but you must find the amount of words you specified while selecting the type of bonus.

When the round starts, you have the 7 hens there, each with a letter. Those letters can form 33 words, with many 3-letter-words to one 7-letter-word. Simply type in the word and if it is a new one, it will be added to the egg rows at the top. If the word is a duplicate, a red box will tell you that. If you don't find anything more, you can either scramble the letters, get a hint (limited amount) or skip to the next level (only if you have revealed enough words). If you proceed to the next round, all missing worlds will be revealed so you can see what you have missed.

The more words you reveal will of course increase your score. Once you run out of time (in time-mode-play), the game ends. In non-timed play you have all the time you want.

Version differences

In Fowl Words you have only one game mode (timed play) and all other modes and special features are locked until you buy the "upgrade". Chicktionary released one year later has all those optional components already unlocked.
