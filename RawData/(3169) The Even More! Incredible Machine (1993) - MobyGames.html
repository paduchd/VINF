Description official description

The same basic gameplay as the original Incredible Machine, solving a puzzle only using the parts provided. It includes all of the 87 puzzles from the first game, 73 new ones, as well as new parts and music.

Trivia

CD-ROM version

There is only one difference between the CD and disk versions. The CD version contains regular CD audio tracks for background music. This audio option isn't required, if any of the supported music boards are selected instead of CD audio for music, then there is no difference between the two.
