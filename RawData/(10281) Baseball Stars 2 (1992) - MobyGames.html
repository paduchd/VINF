Description

The second in the Baseball Stars series, Baseball Stars 2 was done by Romstar. Contains a more polished baseball game at the expense of a few of the customizing features that the first Baseball Stars had. The returning features include a create-a-team mode, upgrading and hiring/firing players, and league mode. Missing features removed for this game include renaming players, renaming teams, and females are no longer present. This sequel contains 12 pre-made teams and 6 slots for user created teams. There are four selectable stadiums to play in this time around.

Trivia

Manual errors

The manual incorrectly states that when increasing attribute points on a player, a random number between 1 and 6 is chosen when an upgrade is purchased. Actually, it is a random number between 1 and 5.

It also states that a hired rookie player has low attribute points and high max attribute points. This was changed to both low attribute points and low max attribute points.
