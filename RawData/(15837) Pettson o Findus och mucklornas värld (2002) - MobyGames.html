Description official description

In the fourth game with Pettson and Findus the cat, you have to guide Findus and a companion through the house. Twelve minigames are scattered over different locations such as the kitchen, the attic, the bedroom and the living room. By completing puzzles, you receive collectible cards, items and finally keys which can be used to unlock hidden worlds.

This installment has much more interactivity. Classic games such as Battleship, The Incredible Machine and Lunar Lander have inspired the developers to create modern spin-offs for kids. Not only reflexes are trained, but maths and logic as well.
