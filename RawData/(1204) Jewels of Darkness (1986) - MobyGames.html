Description

This is a collection of three previously-published Rainbird interactive fiction (with graphics) games. It contains:* Adventure Quest

  • Colossal Adventure
  • Dungeon Adventure

The J.R.R. Tolkien Middle-earth theme was removed in this compilation. Only the respective original releases of the games have references to Middle-earth.

Trivia

Sinclair QL port

Level 9 announced the game also for the Sinclair QL, and loading instructions for such a version can be found in manuals of the existing European releases. The port is however absent from later issued price lists, and also seems to be unknown in QL circles, thus suggesting it was cancelled.
