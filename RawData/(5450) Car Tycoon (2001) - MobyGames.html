Description official description

Car Tycoon is a manager game in which you are the owner of a car company that produces and sells cars. You start in the 1950's in a small town somewhere in the USA. You build a factory, a car shop and a repair facility (most of your money is gone by then) and start to produce cars. By selling them, you'll get money to invent new parts which will then power your new car models. There are up to 3 computer players which want to become rich, too.

Features:* develop more than 400 cars, 7 different types

  • 40 scenarios
  • more than 400 pre-made cars from 1950-2000
  • more than 300 different buildings
  • Zoom

Car Tycoon: Good Economy, Poor Performance, Quality Issues

The Good
Car Tycoon is easy to learn, and provides an interactive tutorial mode to learn the game's interface. Graphically, the game isn't very special.

The Bad
The game is a poor simulation, staging automotive competition within the boundaries of a small city or two. Dealerships can only sell from 2 to 4 different models. The game has some annoying bugs, such as the glitch that occasionally has dealers stocking discontinued models, preventing new models from being sold. The load time is S-L-O-W, even on fast machines.

The Bottom Line
This game gives the user the impression that he/she is using a beta product. Error messages are in German. Bugs affect playability. The maker's web site offers no support information. I truly wished that Maxis, the Sim people, would make something like a "SimCar", or that the maker's of the old game "Detroit" would offer an updated version with the old "global" layout.

Windows · by Kevin Forth (1) · 2002

Trivia

Development

The game is programmed entirely in Borland Delphi, not C/Assembly as most other games.

Multiplayer

The initial German release box contains a short pamphlet telling the poor buyer that the advertised multi-player mode isn't even included "because of release date concerns". They tell that they will patch it later.
