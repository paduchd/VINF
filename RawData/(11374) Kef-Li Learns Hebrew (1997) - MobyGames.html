Description

Kef-Li Learns Hebrew is a game where an affable pink blob by the name of Kef-Li teaches you Hebrew, despite the fact that that conflicts with the title....

A bit of story goes like this; you are spending some time at one Ramah camp. Once at this camp Kef-Li discovers a treasure chest with, not one but, six keyholes. Oddly enough there are six areas in the camp...

Kef-Li informs you that each area has a secret key hidden in it that can only be found by learning Hebrew (?) So off you go to examine the areas. In each of these areas there will be objects to click on, each of which divulges it's Hebrew name. Once you've clicked on all of them at test shows up (Often multiple choice) Succeed to get that area key.

The game seems to actually be trying to improve the knowledge of individuals who already speak the language a little, as most of what you get from the game is vocabulary words.
