Description official description

As it name suggests, Biathlon 2002 simulates biathlon. This sport consists of cross-country skiing and rifle shooting on a shooting range. The character's energy needs to be spent wisely, as skiing constantly at a very fast pace will impede accuracy on the shooting part.

There are about 15 different athletes to choose from and various courses to race on. Everything is presented in real-time 3D graphics. According to the publisher, this was the first biathlon game on PC at the time of release.

To Infogrames: please check if Quality Control Dept. is still alive!

The Good
Nothing.

The Bad
Everything.

It starts that I need to install some obscure driver (not DirectX) to run the game. I don't like to install drivers just for one game. Then, the graphics are absolutely NOT realistic (as the ad blurb may suggest) - they are low standard 3D, with jerky animations. Everything has a VERY low polygon count so everything looks like a triangle :)

The controls are not from this world - it's simply impossible NOT to run into some track boundary, where you will stand forever because the collision detection doesn't work too well... your athlete just hangs on a triangle :)

The sound... uh, never mind the sound.

When I entered the credits, I couldn't find any playtesters or QA people. I can imagine why - it's probably because no-one tested this game :) It must be like that, otherwise the game would have not been released at all.

The Bottom Line
Just let it sit in your dealer's rack forever. Hey, he probably can send it back to Infogrames anyway, so do yourself a favor and DON'T BUY IT!
